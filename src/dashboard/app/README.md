# App Directory

Next.js 14 App Router pages and layouts.

- `layout.tsx` - Root layout with providers (HeroUI, themes, QueryClient)
- `page.tsx` - Dashboard home with KPI cards
- `providers.tsx` - Client-side provider composition
- `globals.css` - Tailwind imports and custom CSS variables
- `proposals/` - Proposal list and management
- `analytics/` - 3D chart visualizations
- `search/` - Vector search interface
