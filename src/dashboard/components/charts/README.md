# Chart Components

Three.js 3D visualizations using React Three Fiber.

- `three-canvas-wrapper.tsx` - Dynamic import wrapper (SSR disabled)
- `status-bar-chart-3d.tsx` - 3D bar chart for proposal status distribution
- `budget-pie-3d.tsx` - 3D pie chart for department budget allocation
- `timeline-chart.tsx` - 3D timeline of proposal submissions
